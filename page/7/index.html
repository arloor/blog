  3. 安装

    2. Centos9 下， 将 ttf 文件夹下的文件全部移动到 /usr/share/fonts/${newdir}下 , yum install -y fontconfig 并执行 fc-cache 。然后执行 fc-list 即可看到新的字体

  5. 搜索 font size，改成 13。如果这个字体还不够大，可以 command + +来放大 UI。

Openssl使用：自定义CA签发证书、私钥格式、s_client使用 May 11, 2023

签发CA并使用CA签发SSL证书

  1. 在 ~/ca 创建CA证书和私钥，私钥为 ca_key.pem ，公钥为 ca.pem 。 ca.pem 后续将被安装到系统并信任。
  2. 在 ~/ca/certs 创建自定义SSL证书，私钥为 privkey.pem ，公钥为 cert.pem。他们将被用于启动https服务。

脚本下载：ca.sh。
