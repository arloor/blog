# 访问[arloor博客](http://www.arloor.com)
使用hugo生成静态博客

## 安装[hugo 0.54 extended](https://github.com/gohugoio/hugo/releases/tag/v0.54.0) （需要支持 scss）

## 部署

```
wget -O /usr/local/bin/tarloor http://www.arloor.com/tarloor.sh
bash tarloor
```
