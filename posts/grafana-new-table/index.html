Pannel Query beforeжҹҘиҜўпјҡ е®ҡд№үPromqlе’ҢжҹҘиҜўoptionдҫӢеҰӮпјҡlegendеҗҚз§°гҖҒMin stepгҖҒFormatе’ҢtypeпјҲRangeгҖҒInstantпјү

Relative timeпјҡ

иҰҶзӣ–dashboardеҸідёҠи§’зҡ„relative timeпјҲеҜ№з»қеҜ№ж—¶й—ҙдёҚз”ҹж•ҲпјүгҖӮдё»иҰҒжңүдёӨз§Қж јејҸпјҡ

  • now-5dжҲ–иҖ…5dпјҡйғҪиЎЁзӨә last 5 day
  • now/d: иЎЁзӨәtoday so farгҖӮ /dеҸҜд»ҘзҗҶи§Јдёәж•ҙйҷӨdayпјҢиҝӣиҖҢеҸҜд»ҘзҗҶи§ЈдёәеҜ№йҪҗеҲ°dayзҡ„ејҖе§ӢпјҢд№ҹе°ұжҳҜеҜ№йҪҗеҲ°00:00
  • now-5d/d: иЎЁзӨәд»Һ5еӨ©еүҚзҡ„00:00еҲ°зҺ°еңЁгҖӮ/dеҜ№йҪҗеҲ°дәҶ00:00

Time shiftпјҡ

time shiftжҳҜе°Ҷrelative timeзҡ„ејҖе§Ӣж—¶й—ҙе’Ңз»“жқҹж—¶й—ҙйғҪеҫҖеүҚеҒҸз§»дёҖж®өж—¶й—ҙпјҢд№ҹеҸӘеңЁж—¶й—ҙйҖүжӢ©жЎҶжҳҜзӣёеҜ№ж—¶й—ҙж—¶з”ҹж•ҲгҖӮдҫӢеҰӮпјҡ

  • 1dпјҡејҖе§Ӣж—¶й—ҙе’Ңз»“жқҹж—¶й—ҙйғҪеҒҸз§»дёҖеӨ©
  • 1d/d: ејҖе§Ӣж—¶й—ҙе’Ңз»“жқҹж—¶й—ҙйғҪеҒҸз§»дёҖеӨ©пјҢз„¶еҗҺеҜ№йҪҗеҲ°00:00
  • 0d/d: е°ҶејҖе§Ӣж—¶й—ҙе’Ңз»“жқҹж—¶й—ҙйғҪеҜ№йҪҗеҲ°00:00гҖӮиҝҷж ·д№ҹе°ұжҳҜе®һзҺ°дәҶtodayгҖӮеҢ…жӢ¬1m/d=0d/dпјҢеӣ дёә1mж•ҙйҷӨdе°ұжҳҜ0dж•ҙйҷӨd

жҖ»зҡ„жқҘи®ІпјҢе…ій”®иҰҒзҗҶи§ЈпјҢ/dгҖҒ/wгҖҒ/yйғҪжҳҜж•ҙйҷӨпјҢж•ҙйҷӨе°ұеҸҜд»ҘзҗҶи§ЈдёәеҜ№йҪҗеҲ°ж•ҙdayгҖҒж•ҙweekгҖҒж•ҙyearгҖӮеҸҜд»ҘиҮӘе·ұеҲ°Playground: Time range override.дҝ®ж”№иҜ•иҜ•зңӢгҖӮ
