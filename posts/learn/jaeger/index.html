OpenTracingд»Ӣз»Қ

jaegerдҪҝз”ЁдәҶOpenTracingзҡ„иҜӯж„Ҹе’Ңзұ»еә“пјҢOpenTracingзҡ„зӣ®зҡ„жҳҜеҲӣе»әдёҖдёӘе№іеҸ°ж— е…іпјҢеҺӮе•Ҷж— е…ізҡ„traceзі»з»ҹж ҮеҮҶгҖӮеңЁд»Ӣз»ҚjaegerеүҚе…ҲиҜҙдёҖдёӢOpenTracingжҠҪиұЎеҮәжқҘзҡ„еҮ дёӘжҰӮеҝөгҖӮ

дёӯж–Үзҝ»иҜ‘

traceз”ұspanз»„жҲҗ

spanеҸҜд»ҘзҗҶи§ЈдёәдёҖдёӘвҖңиҝҮзЁӢвҖқпјҢеңЁеӣҫзүҮдёӯз”ЁдёҖдёӘжңүи·ЁеәҰзҡ„зәҝпјҲй•ҝж–№еҪўпјүиЎЁзӨәпјҢй•ҝеәҰеҚіж¶ҲиҖ—зҡ„ж—¶й—ҙгҖӮ

traceеҲҷз”ұеӨҡдёӘspanз»„жҲҗгҖӮе°ҶеӨҡдёӘзӢ¬з«Ӣзҡ„spanиҒ”зі»иө·жқҘзҡ„зҡ„жҳҜrequestIDпјҢд№ҹе°ұжҳҜspanзҡ„дёҠдёӢж–ҮдҝЎжҒҜгҖӮspanе’Ңspanд№Ӣй—ҙжңүдёӨз§Қе…ізі»пјҡchildOfе’ҢFollowsFromгҖӮдёҠеӣҫзҡ„е·Ұиҫ№е°ҶеҮҪж•°и°ғз”Ёй“ҫжҸҸиҝ°жҲҗдёҖдёӘж ‘пјҢchildOfе°ұжҳҜж ‘дёӯзҡ„зҲ¶еӯҗе…ізі»пјҢfollowsFromе°ұжҳҜе…„ејҹе…ізі»гҖӮ

spanеҢ…еҗ«зҡ„дҝЎжҒҜпјҡ

  • An operation name
  • A start timestamp
  • A finish timestamp
  • A set of zero or more key:value Span Tags. The keys must be strings. The values may be strings, bools, or numeric types.
  • A set of zero or more Span Logs, each of which is itself a key:value map paired with a timestamp. The keys must be strings, though the values may be of any type. Not all OpenTracing implementations must support every value type.
  • A SpanContext (see below)
  • References(childOf/followsFrom) to zero or more causally-related Spans (via the SpanContext of those related Spans)

SpanContextеҢ…еҗ«дёӨйғЁеҲҶдҝЎжҒҜпјҡ

  1. иғҪе®ҡдҪҚеҲ°дёҖдёӘзү№е®ҡspanзҡ„зҠ¶жҖҒдҝЎжҒҜпјҲtraceIdе’ҢspanIdпјү
  2. еңЁиҝӣзЁӢд№Ӣй—ҙдј йҖ’зҡ„Baggage ItemsпјҲеҲҶеёғејҸtraceйңҖиҰҒдёҠдёӢж–ҮдҝЎжҒҜпјү

дҪҝз”Ёд»ҘеҸҠд»Јз Ғдҫөе…ҘеһӢеҲҶжһҗ

зӨәдҫӢд»Јз Ғ иҝҷд»Ҫд»Јз Ғеұ•зӨәдәҶеҰӮдҪ•еңЁеҚ•иҝӣзЁӢдёӯдҪҝз”ЁJaeger ClientгҖӮеҸҜд»ҘзңӢеҲ°еҲӣе»әдәҶдёҖдёӘеҚ•дҫӢзҡ„TracerеҜ№иұЎпјҢе®ҡд№үеҘҪиҜҘTraceеҰӮдҪ•иҝӣиЎҢйҮҮж ·пјҢд»ҘеҸҠеҰӮдҪ•еҗ‘agentдёҠжҠҘж•°жҚ®пјҲHttpгҖҒrpcгҖҒudpпјү

еңЁеҫ®жңҚеҠЎзҺҜеўғдёӢпјҢж¶үеҸҠtraceзҡ„и·ЁиҝӣзЁӢдј ж’ӯпјҢйҰ–е…ҲеҸҜд»ҘеңЁControllerдёҠеўһеҠ еҲҮйқўпјҢжҠҪеҸ–SpanContextдҝЎжҒҜпјҢе…¶ж¬ЎеҸҜд»ҘеҜ№еҗ‘еӨ–еҸ‘йҖҒиҜ·жұӮзҡ„еҜ№иұЎиҝӣиЎҢadaptпјҢдҫӢеҰӮеҜ№HttpClientиҝӣиЎҢд»ЈзҗҶпјҢеўһеҠ зӣёе…іHeaderд»ҘInject SpanContextдҝЎжҒҜгҖӮеҸҰеӨ–пјҢеӣ дёәTracerеҜ№иұЎжҳҜеҚ•дҫӢзҡ„пјҢжүҖд»ҘTracerеҸҜд»ҘдҪңдёәSpringзҡ„дёҖдёӘBeanиҝӣиЎҢеҲқе§ӢеҢ–е’ҢдҪҝз”ЁгҖӮ
