OpentelemetryжҳҜжҖҺд№ҲеҒҡй“ҫи·ҜиҝҪиёӘзҡ„д»Ӣз»ҚдәҶopentelemetryзҡ„APIе’ҢSDKе®һзҺ°пјҢд№ҹд»Ӣз»ҚдәҶеҰӮдҪ•иҝӣиЎҢжүӢеҠЁд№°зӮ№гҖӮиҝҷзҜҮеҚҡе®ўжҳҜд»Ӣз»ҚеҰӮдҪ•иҝӣиЎҢиҮӘеҠЁеҹӢзӮ№пјҢиҝҷеҪ“з„¶иҰҒз”ЁеҲ°javaagentжҠҖжңҜдәҶгҖӮJava Agentе®һзҺ°жҢҮеҚ—жңүеҶҷиҝҮjavaagentзҡ„е®һзҺ°жҢҮеҚ—е’Ңзұ»еҠ иҪҪзҡ„еқ‘пјҢд»ҠеӨ©зңӢзңӢopentelemetryжҳҜеҰӮдҪ•еҒҡзҡ„пјҢе…¶githubең°еқҖжҳҜopentelemetry-java-instrumentationпјҢжҺҘдёӢжқҘеә”иҜҘе°ұжҳҜи·ҹзқҖд»Јз ҒжөҒж°ҙиҙҰдәҶгҖӮ

OpenTelemetryAgentзұ»

иҝҷжҳҜjavaagentзҡ„Premain-classе’ҢAgent-classпјҢд№ҹжҳҜjavaagentзҡ„еҗҜеҠЁзұ»гҖӮжіЁйҮҠдёӯеҶҷеҲ°пјҢиҜҘзұ»дјҡиў«SystemClassLoaderеҠ иҪҪпјҢе…¶д»–зұ»дјҡиў«BootstrapClassLoaderеҠ иҪҪгҖӮеҰӮжһңеңЁе…¶д»–зұ»дёӯеҶҚеҸҚеҗ‘еј•з”ЁиҜҘзұ»пјҢеҲҷдјҡеҜјиҮҙиҜҘзұ»еҶҚиў«BootStrapClassLoaderеҠ иҪҪпјҢеҜјиҮҙеҮәзҺ°дёӨдёӘдёҚеҗҢзҡ„OpenTelemetryAgentзұ»пјҲеӣ дёәзұ»еҠ иҪҪеҷЁдёҚеҗҢпјүгҖӮдёәдәҶйҳІжӯўиҝҷд»¶дәӢеҸ‘з”ҹпјҢжіЁйҮҠйҮҢжҸҗдәҶеҮ дёӘжіЁж„ҸзӮ№пјҡ

  1. иҜҘзұ»е°ҪеҸҜиғҪеҒҡе°‘зҡ„дәӢжғ…
  2. з»қеҜ№дёҚиҰҒд»Һе…¶д»–зұ»еј•з”ЁиҜҘзұ»
  3. дёҚиҰҒеңЁиҜҘзұ»дёӯеӯҳеӮЁstaticж•°жҚ®пјҲеҰӮжһңзңҹеҮәзҺ°дёӨдёӘзұ»зҡ„жғ…еҶөпјҢйӮЈд№Ҳе…¶е®һжҳҜдёӨд»Ҫstaticж•°жҚ®пјү
  4. дёҚиҰҒеңЁиҜҘзұ»дёӯеҲқе§ӢеҢ–д»»дҪ•ж—Ҙеҝ—з»„д»¶гҖӮ

йӮЈд№ҲиҜҘзұ»еҒҡдәҶе“ӘдәӣдәӢе‘ўпјҹ

  1. жӢҝеҲ°agentJarпјҢе№¶е°ҶиҜҘjarи®ҫзҪ®еҲ°BootStrapClassLoaderзҡ„жҗңзҙўи·Ҝеҫ„пјҲдёәдәҶз”ұbootstrapClassLoaderеҠ иҪҪе…¶д»–зұ»пјү
  2. е°ҶinstrumentionеҜ№иұЎж”ҫзҪ®еҲ°Holderзҡ„зұ»дёӯпјҢд»ҘдҫҝеңЁе…¶д»–ең°ж–№дҪҝз”Ё
  3. и°ғз”ЁAgentInitializerпјҢиҝӣе…ҘзңҹжӯЈзҡ„agentеҲқе§ӢеҢ–е·ҘдҪңгҖӮ

жӯҘйӘӨ1е’Ң2еә”иҜҘжҳҜжүҖжңүjavaagentйғҪйңҖиҰҒзҡ„е·ҘдҪңпјҢиҮӘе·ұе®һзҺ°agentд№ҹеҸҜд»ҘеҖҹйүҙгҖӮз¬¬3жӯҘжҳҜopentelemetryзҡ„ж ёеҝғдәҶпјҢжҺҘдёӢжқҘиҝӣе…ҘиҝҷдёҖйғЁеҲҶгҖӮ

AgentInitializerзұ»

иҝҳжҳҜе…ҲзңӢжіЁйҮҠпјҡиҝҷдёӘзұ»иў«OpenTelemetryAgentи°ғз”ЁпјҢеӣ жӯӨз”ұbootstrapClassLoaderеҠ иҪҪпјҢеӣ жӯӨеҸҜд»Ҙз•…йҖҡең°дҪҝз”Ёagentзҡ„е…¶д»–зұ»пјҲеӣ дёәе…¶д»–зұ»еҗҢж ·иў«bootstrapClassLoaderеҠ иҪҪпјү

иҝҷд№ҹз¬ҰеҗҲеңЁдёҠйқўжҸҗеҲ°зҡ„пјҢеңЁpremainе’Ңagentmainзҡ„зұ»дёӯеҒҡе°ҪеҸҜиғҪе°‘зҡ„дәӢ

иҝҷдёӘзұ»е®һйҷ…дёҠеҒҡдәҶдёүд»¶дәӢпјҡ

  1. еҲӣе»әAgentClassloader
  2. дҪҝз”ЁAgentClassloaderеҠ иҪҪAgentStarterImplзұ»
  3. и°ғз”ЁAgentStarterImplиҝӣиЎҢagentзҡ„еҲқе§ӢеҢ–пјҲиҜҘзұ»жүҖеҠ иҪҪзҡ„жүҖжңүзұ»йғҪжҳҜз”ұAgentClassLoaderеҠ иҪҪзҡ„пјҢиҜҘзұ»зҡ„жүҖжңүжҙ»еҠЁйғҪеңЁAgentClassLoaderзҡ„иҢғеӣҙеҶ…пјү

AgentClassloader

otelзҡ„classloaderж•ҙдҪ“жҖқжғіжҳҜзӣ®ж ҮjarеңЁagent.jarдёӯпјҢзӣ®ж ҮjarдёӯйғҪжҳҜ.classdataж–Үд»¶пјҢд»ҘзЎ®дҝқдёҚдјҡиў«е…¶д»–зұ»еҠ иҪҪеҷЁеҠ иҪҪеҲ°гҖӮ

иҜҘclassloaderзҡ„зҲ¶дәІзұ»еҠ иҪҪеҷЁеңЁjdk8еҸҠд»ҘдёӢжҳҜnull(BootstrapClassLoader)пјҢеңЁjdk9еҸҠд»ҘдёҠжҳҜPlatformClassLoaderпјҲзӣ®жөӢеә”иҜҘзұ»дјјBootstrapClassLoaderпјүгҖӮ

йҷӨжӯӨд№ӢеӨ–пјҢиҝҳжңүдёҖдәӣз»ҶиҠӮеңЁд»Јз ҒдёӯпјҢдҫӢеҰӮ

  1. дёҚиғҪеңЁAgentClassLoaderдёӯдҪҝз”Ёslfj
  2. дёәдәҶиҺ·еҸ–agent.jarдёӯзҡ„resourceпјҢйңҖиҰҒдёҖдёӘproxyеҜ№иұЎ
  3. й’ҲеҜ№promtheusзҡ„exporterзұ»дҪҝз”ЁеҲ°зҡ„jdk httpserverеҒҡдәҶеҠ иҪҪзҡ„жҺ’йҷӨ
  4. еҜ№otelиҮӘиә«з”ЁеҲ°зҡ„grpcзұ»еҒҡдәҶжҺ’йҷӨ

AgentStarterImpl

жһ„йҖ еҰӮдёӢпјҢж¶үеҸҠеҲ°дёҖдёӘзұ»еһӢејәиҪ¬пјҡ

  private static AgentStarter createAgentStarter(
      ClassLoader agentClassLoader, Instrumentation instrumentation, File javaagentFile)
      throws Exception {
    Class<?> starterClass =
        agentClassLoader.loadClass("io.opentelemetry.javaagent.tooling.AgentStarterImpl");
    Constructor<?> constructor =
        starterClass.getDeclaredConstructor(Instrumentation.class, File.class);
    // иҝҷйҮҢжңүзұ»еһӢејәиҪ¬еҲ°interface
    return (AgentStarter) constructor.newInstance(instrumentation, javaagentFile);
  }

иҝҷйҮҢеҲҶжһҗдёӢдёәд»Җд№ҲиҝҷдёӘзұ»еһӢејәиҪ¬жІЎжңүйҒҮеҲ°зұ»еҠ иҪҪй—®йўҳпјҹдёҚиҰҒи§үеҫ—иҝҷйҮҢе•°е—ҰпјҢиҮӘеҠЁеҹӢзӮ№зҡ„йҡҫзӮ№еҸӘжңүдёӨдёӘпјҢдёҖдёӘжҳҜжҖҺд№ҲеҒҡеӯ—иҠӮз Ғдҝ®ж”№пјҢеҸҰдёҖдёӘе°ұжҳҜеӨ„зҗҶзұ»еҠ иҪҪйҡ”зҰ»гҖӮ

AgentStarterImplжҳҜAgentClassLoaderеҠ иҪҪзҡ„пјҢд»–жҳҜAgentStarterиҝҷдёӘжҺҘеҸЈзҡ„е®һзҺ°зұ»пјҲз”ұBootstrapClassloaderеҠ иҪҪпјүгҖӮеӣ дёәAgentStartImpl extends(impl) AgentStartImpl, дё” AgentClassLoader extends BootstrapClassLoaderпјҲйҖ»иҫ‘дёҠзҡ„зҲ¶еӯҗе…ізі»пјүгҖӮ

AgentStarterImplеҸҲеҲӣе»әдәҶеҗ„з§ҚжҸ’д»¶зҡ„extensionClassLoaderпјҢзӢ¬з«Ӣclassloaderзҡ„зӣ®зҡ„жҳҜдёәдәҶйҒҝе…ҚдёҺз”ЁжҲ·д»Јз Ғзҡ„еҪұе“ҚгҖӮ

жңҖеҗҺе°ҶextensionClassLoaderи®ҫзҪ®дёәзәҝзЁӢдёҠдёӢж–Үзҡ„ClassloaderпјҢе№¶е®үиЈ…BytebuddyAgentжқҘеҒҡеӯ—иҠӮз Ғдҝ®ж”№гҖӮ

  @Override
  public void start() {
    extensionClassLoader = createExtensionClassLoader(getClass().getClassLoader(), javaagentFile);
    ClassLoader savedContextClassLoader = Thread.currentThread().getContextClassLoader();
    try {
      Thread.currentThread().setContextClassLoader(extensionClassLoader);
      AgentInstaller.installBytebuddyAgent(instrumentation);
    } finally {
      Thread.currentThread().setContextClassLoader(savedContextClassLoader);
    }
  }
